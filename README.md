# 📊 Customer Churn Intelligence System
## 📋 Project Overview
This project focuses on predicting customer attrition using machine learning techniques. By analyzing behavior patterns and contract details, the system identifies high-risk customers, enabling businesses to implement proactive retention strategies.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Algorithms:** Logistic Regression & Decision Tree Classifier

## 🚀 Key Insights & Results
- **Accuracy:** The model achieved an overall accuracy of **79%**.
- **Top Predictors:** Identified **Tenure** and **Monthly Charges** as the strongest indicators of churn.
- **Business Impact:** Helps marketing teams target specific customer segments (like those with month-to-month contracts) to reduce losses.

## 📁 How to Use
1. Open the `Customer-Churn-Intelligence-System.ipynb` to view the full analysis and code.
2. The notebook includes data cleaning, visualization, and model evaluation steps.
